# Weather MCP Server

A simple MCP server that provides weather data for cities using the Open-Meteo API.

## Usage

```bash
npm start
```

Or directly with tsx:

```bash
npx tsx main.js
```

## Tool

- `get-weather` - Get current weather for any city
