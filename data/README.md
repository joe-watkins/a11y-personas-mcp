# Data

This folder contains the personas data for the MCP server.

## Structure

- **`a11y-personas/`** - Git submodule pointing to [joe-watkins/a11y-personas](https://github.com/joe-watkins/a11y-personas)
- **`personas.json`** - Built personas data (generated by `npm run build`)

## Updating Personas

To pull the latest personas from the source repository:

```bash
npm run update-personas
```

This will:
1. Pull latest changes from the submodule
2. Rebuild `personas.json` from the submodule's markdown files
